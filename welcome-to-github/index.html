#Getting Started

If you are new to github then this is the right place to kickstart your github journey.

You’ll learn how to :

  • Create and use a repository
  • Make changes to a file and push them to GitHub as commits
  • Open and merge a pull request

What is Github???

"GitHub is a web-based Git or version control repository and Internet hosting service" This is what wikipedia says!! Let me make it simple,Github is a place where you host or upload opensource project so that others can use it and even help us in adding more features to it.

Requirements

A github account,Internet Access!!

Lets Begin..

1) Create a Repository

A repository is usually used to organize a single project.Repository can be considered as a folder where you keep the files of a particular project.

To Create A New Repository-:

  • In the upper right corner, next to your avatar, click and then select New repository.
  • Name your repository hello-world.
  • Write a short description.
  • Select Initialize this repository with a README.

README is a file where you can give information about your project like installation of the project or how it works or anything related to the project.The README file will have a great affect on the project.

2) Create New File/Upload/Edit and commit changes

Once you create a new repo(repository),your window will look like this..

To create a new file :

  • click on the Create new File

  • Enter The required name with some extension(.txt , .c , .js etc..) and add the content.

  • Then go down and at the end There will be a box named commit changes(Whatever new files or any changes that you make it should be updated in the repository , for that we need to commit our changes).So give a commit message(Mentioning the change that you made) and click on Commit Changes/New file

To upload a new file :

  • click on Upload files and then select your file.
  • Then add a commit message (like "add image/code/whatever..")
  • Click on commit changes

To Edit files :

  • Lets take the readme file as an example. Click on the README.md(md stands for MarkDown) file
  • Click on the "pencil" icon and editor will open and make the required changes
  • add a commit message(Like edit README) and click on commit changes
